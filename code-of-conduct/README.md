# Verana Council — Code of Conduct

> **Status:** Pre-ratification draft placeholder. Amended by the same process as
> the [Bylaws](../bylaws/).

Conduct rules binding on all Council participants.

## Outline (draft)

1. Principles (neutrality, transparency, good faith)
2. Conflict-of-interest cooldown (recusal on decisions affecting direct
   competitors)
3. Seat-diversity rule (sector spread with soft regional balance, per the
   Council seat model)
4. Confidentiality of closed-session matters
5. Reporting and enforcement (graduated sanctions via the
   [Network GF](../network-gf/) rubric)
6. Appeals
